# Time Complexity

## Types

O(1) constant
O(logn) logaritmic // usually when we cut in half
O(n) linear
O(n^2) quadratic
O(k^n) exponential

---

## Examples

arr.push() // O(1)
arr.pop() // O(1)
1+3 // O(1)
for(...) // O(n)
arr.unshift // O(n)

for(...)
    1+1
O(n)

for(...)
    for(...)
        1+1
O(n^2)

for
    for
        for
O(n^3)
